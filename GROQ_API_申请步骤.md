## 什么是 Groq？
Groq 提供极速 LLM API（比 GPT-4 快 10x），有**每天免费额度**，不需要信用卡，完全免费注册。

---

## 第一步：注册账号

1. 打开 **https://console.groq.com**
2. 点击 **"Sign up"**
3. 用 Google 账号 或 Email 注册（推荐 Google，更快）
4. 验证邮箱

---

## 第二步：创建 API Key

1. 登录后，点击左侧菜单 **"API Keys"**
2. 点击 **"+ Create API Key"**
3. 名称填：`gamead-insight`（随便填也可以）
4. 点击创建
5. **复制生成的 Key**（以 `gsk_` 开头，**只显示一次**，务必保存好）

---

## 第三步：把 Key 配置到平台

有两种方式：

### 方式A：直接告诉我（推荐）
把复制的 API Key（以 `gsk_` 开头的一串字符）发给我，我帮你配置到后端。

### 方式B：自己配置
1. 在 `hotspot-v3/backend/` 目录下新建文件 `.env`
2. 写入内容：
```
GROQ_API_KEY=gsk_你的key内容
```
3. 重启后端
